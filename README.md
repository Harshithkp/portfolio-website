# Harshith K P Portfolio

Premium responsive portfolio website built with HTML, CSS and JavaScript.

## Sections
- Hero
- About
- Skills
- Projects
- Experience
- Education
- Contact

## Deploy on Vercel
1. Upload this folder to GitHub.
2. Open Vercel and import the GitHub repository.
3. Keep default settings and click Deploy.

## Customize
- Add project GitHub/live links inside `index.html`.
- Replace initials avatar with your profile image if needed.
- Update colors in `style.css` under `:root`.
